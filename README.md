# smart-recipe-finder
A modern desktop recipe finder built using CustomTkinter and the Spoonacular API.
This application allows users to enter ingredients they have and fetches live recipes with images, used/missing ingredients, and full recipe details.

Features:
Search recipes using ingredients
Live recipe images
Shows used & missing ingredients
Detailed recipe popup
Open full recipe in browser
Modern dark UI with CustomTkinter

Built With:
Python
CustomTkinter
Spoonacular API
Pillow (PIL)
Requests
